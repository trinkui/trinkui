# @trinkui/shared

Shared TypeScript types and constants used internally by the [trink-ui](https://github.com/trinkui/trinkui) component library.

> **Note:** This is an internal package. You probably want [`@trinkui/react`](https://www.npmjs.com/package/@trinkui/react) instead.

## What's Inside

- **Types** — `ActionProps`, `ImageProps`, `Theme`, `Size`, `BaseSectionProps`, `SectionHeaderProps`, and more
- **Constants** — `CONTAINER_SIZES`, `SECTION_SPACING`, animation defaults

## Usage

```ts
import type { BaseSectionProps, ActionProps, Theme } from "@trinkui/shared";
```
