# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - 2026-03-18

### Added
- Initial release with 49+ components
- 38 primitive components (Button, Input, Modal, Tabs, Table, Toast, etc.)
- 11 section components (Hero, Features, Pricing, CTA, etc.)
- 5 animation wrappers (FadeIn, SlideUp, StaggerChildren, ScaleIn, BlurIn)
- 3 layout components (Container, Section, SectionHeader)
- 8 landing page templates (SaaS, Agency, Product Launch, AI, Mobile, Startup, Portfolio, Event)
- Dark/light mode support via CSS variables
- Full TypeScript support with strict mode
- Documentation site with 90+ pages
- Customization guide (theming, colors, dark mode, variants)
